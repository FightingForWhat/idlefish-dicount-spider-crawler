闲鱼折扣爬虫

--------------------------------

闲下来了，突然想做一个程序，看看闲鱼出售的东西谁的折扣低。

自己开发了闲鱼接口，然后接入。从响应的数据中抓取价格、原价、折扣信息。

继续开发的话，将自动做折扣排序，再做产品对比，按折扣力度排序的话，理论上就可以拿到折扣最低的商品了。
